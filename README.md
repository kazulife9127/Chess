# ブラウザ同士のオンライン対戦ができるChessゲーム

## 概要
- オンライン対戦ができるChessゲーム
- ルームの数は3つまで
- Hololens2とブラウザで対戦も可能(ブランチを「Main」から「Hololens」に変えると、サンプルがあります)
